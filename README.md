# 🥗 Deula – Eat Healthy, Live Better

**Deula** is a Flutter-based health & wellness app that helps users track meals, monitor their daily nutrition, and stay hydrated — all in a beautifully designed, responsive interface.

---

## 🚀 Features

- 🍽️ **Add Meals**: Easily log meals with name, time, and macros (calories, protein, fat, carbs)
- 📊 **Daily Nutrition Summary**: Get a clear overview of your daily intake
- 💧 **Water Intake Tracker**: Set hydration goals and track your water consumption
- 🔒 **Offline Support**: All data is stored locally using SQLite (`sqflite`)
- 🌐 **Localization Support**: Multilingual with `easy_localization`
- 📱 **Responsive UI**: Clean and adaptive layout with `flutter_screenutil`
